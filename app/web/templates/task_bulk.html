批量导入任务

使用方法

  1. 准备一个 UTF-8 编码的 CSV 文件，每行一个任务。
  2. 必填列：sku_id, creative_id, segment_id, video_prompt, target_count + 素材列（看下表）
  3. 可选基础列：sequence_index, depends_on_task_id, max_retry_count, asset_kind
  4. 素材列怎么填（按 asset_kind 选其一，不能混填）：
    asset_kind 用哪些列 说明
    reference（默认） source_asset_path 参考图，多张用 | 分隔
    frames_pair source_start_path + source_end_path 起止帧对（自动设 subtab=frames）
    first_frame source_start_path 仅起始帧（自动设 subtab=frames）

    ⚠ 写错列会被当成错误行拒绝。比如 asset_kind=first_frame 但图片写在 source_asset_path 里，导入时会报错并跳过那一行（不会静默上传到错的槽位）。

  5. 可选生成参数列（每行可独立覆盖账号默认值，留空 = 用账号默认）：
    • mode_model：例 Veo 3.1 - Fast / Veo 3.1 - Quality
    • mode_output_count：1 / 2 / 3 / 4
    • mode_duration_sec：4 / 6 / 8
    • mode_aspect：9:16 / 16:9 / 1:1
    • mode_tab、mode_subtab（一般无需填，frames_pair / first_frame 自动设 subtab）
  6. 所有路径只填文件名（例如 image_001.png），不要写完整路径。
  7. 同时把所有用到的图片一起上传到下面的"所有用到的图片"输入框。

CSV 里的 source_asset_path 会按文件名匹配这里上传的图片。

取消

预览检查会扫描每一行，列出错误（不会创建任务）； 确认无误后再点 "确认导入"。
